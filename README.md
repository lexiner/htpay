<h1>Htpay 1.0</h1>

<h1>安装</h1>

composer环境的安装命令如下

``` bash
composer require lexiner/htpay
```

理论上，支持php5.6+，建议使用7.0版本。
微信、支付宝借鉴dedemao的单文件版，无依赖，方便快速集成。后续会继续加入更多的支付平台。
> 此项目，只为webman设计。其他框架，暂未适配。如有需要，可自行修改。或者等待后续会适配。
